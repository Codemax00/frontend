Asynchronous Programming

Learn how to handle asynchronous operations in JavaScript using callbacks, async/await, and more.

Async/Await Example

async function fetchData() {
    try {
        const response = await fetch('https://api.example.com/data');
        const data = await response.json();
        return data;
    } catch (error) {
        console.error('Error:', error);
    }
}

Promise Example

const myPromise = new Promise((resolve, reject) => {

Classes and Object-Oriented Programming

Learn about ES6 classes and OOP concepts in JavaScript.

Class Example

class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    greet() {
        return `Hello, I'm ${this.name} and I'm ${this.age} years old.`;
    }
}

const person = new Person('John', 30);
console.log(person.greet());

Understanding JavaScript modules and how to use them.

Module Example

export const add = (a, b) => a + b;
export const subtract = (a, b) => a - b;

// main.js
import { add, subtract } from './math.js';
console.log(add(5, 3));

Closures

Understanding closures and their practical applications.

Closure Example

function createCounter() {
    let count = 0;
    return {
        increment: () => ++count,
        decrement: () => --count,
        getCount: () => count
    };
}

const counter = createCounter();
console.log(counter.increment());
console.log(counter.increment());
console.log(counter.getCount());

Prototypes and Inheritance

Understanding JavaScript's prototype-based inheritance system.

Prototype Example

function Animal(name) {
    this.name = name;
}

Animal.prototype.speak = function() {
    return `${this.name} makes a sound.`;
};

function Dog(name) {
    Animal.call(this, name);
}

Dog.prototype = Object.create(Animal.prototype);
Dog.prototype.constructor = Dog;

const dog = new Dog('Rex');
console.log(dog.speak());
